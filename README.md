# LXYZ
A project aiming at HUAWER IOT competition
We use CentOS server as cloud server, acting as a transit station， which has been deployed with modules like django, ftp(for transmitting files), socket(for transmitting instruction), also a part waiting to be done—— the butt with the APP(client).
For the part of hardware, Raspberrypi 4b is used as the Host Computer, while the stm32 microcontroller used as Lower Computer.
